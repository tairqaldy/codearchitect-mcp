# Quick Onboarding

## Install (2 minutes)

1. **Install Node.js**: [nodejs.org](https://nodejs.org/) (v18+)
2. **Install package**: `npm install -g codearchitect-mcp`
3. **Configure IDE**: See [README](../README.md#3-configure-in-your-ide)

## First Use

1. **Open your project** in Cursor or VS Code
2. **Ask AI**: `"use codearchitect"`
3. **Save a conversation**: `"use codearchitect store_session"`
4. **Get it back**: `"use codearchitect get_session"`

That's it! Sessions save automatically to main `.codearchitect/` folder in your home directory (`~/.codearchitect/sessions/`). No workspace configuration needed!

## Tips

- **Store important discussions** - Architecture decisions, complex solutions, design patterns
- **Retrieve when needed** - Don't re-explain, just get the session
- **Organized by date** - Easy to find sessions
- **Topic folders** - Each session gets its own folder
